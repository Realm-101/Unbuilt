# Interactive AI Conversations - User Guide

## Overview

The Interactive AI Conversations feature transforms Unbuilt from a one-shot analysis tool into an interactive AI advisor. After receiving your gap analysis, you can engage in follow-up conversations to explore your idea more deeply, ask clarifying questions, and refine your analysis based on new assumptions.

## Getting Started

### Accessing Conversations

1. **Complete a Gap Analysis**: Run a search to generate your initial gap analysis
2. **Scroll to Conversation Interface**: Below your analysis results, you'll find the conversation interface
3. **Start Asking Questions**: Use the input field or click suggested questions to begin

### Your First Conversation

When you first view an analysis, you'll see:
- **Suggested Questions**: 3-5 AI-generated questions tailored to your analysis
- **Conversation Thread**: Empty initially, fills as you interact
- **Input Field**: Type your own questions here

**Example First Questions:**
- "What evidence supports the market demand for this opportunity?"
- "Why haven't existing competitors addressed this gap?"
- "What should be my first step to validate this idea?"

**‚úÖ Good Questions:**

### Viewing Conversation History

- **Chronological Order**: Messages appear in time order
- **Auto-Scroll**: Automatically scrolls to latest message

**User Messages:**

**When to Clear:**
- Starting a completely new line of inquiry
- Conversation became too long or unfocused

**How to Clear:**

## Analysis Refinement & Variants

### What Are Variants?

Variants are alternative versions of your analysis based on modified assumptions. For example:
- "What if I target the EU market instead of US?"
- "What if I use a subscription model instead of one-time purchase?"
- "What if I focus on enterprise customers instead of consumers?"

### Creating Variants

**Automatic Detection:**
The AI detects when you're requesting a refinement:
- "What if I targeted small businesses instead?"
- "How would this change for the healthcare industry?"
- "What if my budget was $50,000 instead of $10,000?"

**Confirmation Process:**
1. AI identifies refinement request
2. Shows confirmation prompt with modified parameters
3. You confirm to generate new analysis
4. New variant is created and linked to original

### Comparing Variants

**Variant Selector:**
- Switch between original and variants
- See modified parameters for each
- Each variant has its own conversation thread

**Side-by-Side Comparison:**
- Innovation scores
- Top gaps identified
- Competitive landscape
- Action plan differences

**Use Cases:**
- Evaluate different market segments
- Compare business models
- Assess geographic markets
- Test different positioning strategies

## Troubleshooting

### Common Issues

**"Rate limit exceeded"**
- **Cause**: Reached question limit for your tier
- **Solution**: Upgrade to Pro for unlimited questions
- **Workaround**: Wait for daily reset (midnight UTC)

**"AI service error"**
- **Cause**: Temporary AI service issue
- **Solution**: Wait a moment and try again
- **Auto-Retry**: System retries automatically (3 attempts)

**"Context too large"**
- **Cause**: Conversation became very long
- **Solution**: AI automatically summarizes older messages
- **Manual**: Clear conversation and start fresh

**"Inappropriate content detected"**
- **Cause**: Message flagged by content filter
- **Solution**: Rephrase question to focus on business analysis
- **Appeal**: Contact support if you believe this was an error

**"Network error"**
- **Cause**: Connection issue
- **Solution**: Check internet connection
- **Auto-Retry**: System retries automatically

### Response Quality Issues

**Response Not Relevant:**
- Rephrase question with more context
- Reference specific parts of analysis
- Break complex questions into smaller ones

**Response Too Generic:**
- Add specific constraints (budget, timeline)
- Mention specific competitors or markets
- Ask for actionable next steps

**Response Contradicts Analysis:**
- Point out the contradiction in follow-up
- Ask AI to clarify based on analysis data
- Report if issue persists

## FAQ

### General Questions

**Q: How many questions can I ask?**
A: Free tier: 5 per analysis. Pro tier: Unlimited. Enterprise: Unlimited.

**Q: Do conversations expire?**
A: No, conversations are saved indefinitely with your analysis.

**Q: Can I share conversations with my team?**
A: Yes, export as PDF or use Enterprise team features.

**Q: Does the AI remember previous analyses?**
A: Each conversation is scoped to one analysis, but you can reference others.

### Technical Questions

**Q: What AI model powers conversations?**
A: Google Gemini 2.5 Pro with custom prompts for business analysis.

**Q: How long do responses take?**
A: Typically 2-5 seconds. Pro users get streaming responses (faster perceived time).

**Q: Can I use conversations offline?**
A: No, conversations require internet connection for AI processing.

**Q: Is there a mobile app?**
A: The web interface is fully mobile-optimized. Native apps coming soon.

### Billing Questions

**Q: Do conversations count toward my search limit?**
A: No, conversations are separate from initial analysis searches.

**Q: What happens if I downgrade from Pro to Free?**
A: Existing conversations remain accessible, but new questions are limited.

**Q: Are there additional charges for conversations?**
A: No, conversations are included in your tier subscription.
