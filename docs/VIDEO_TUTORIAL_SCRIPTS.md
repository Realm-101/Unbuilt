# Video Tutorial Scripts

This document contains scripts and guidelines for creating video tutorials for Unbuilt's UX features.

## Video Production Guidelines

### Technical Specifications
- **Resolution**: 1920x1080 (1080p)
- **Frame Rate**: 30fps
- **Format**: MP4 (H.264)
- **Audio**: Clear voiceover, background music at -20dB
- **Length**: 2-5 minutes per video (keep concise)
- **Captions**: Include closed captions for accessibility

### Recording Setup
- Use screen recording software (OBS Studio, Camtasia, or ScreenFlow)
- Record in a quiet environment
- Use a quality microphone
- Show cursor movements clearly
- Highlight clicks and interactions
- Use zoom-in effects for small UI elements

### Branding
- Include Unbuilt logo in corner
- Use brand colors (purple, red, orange)
- Add intro/outro animations (3-5 seconds)
- Include call-to-action at end

## Video Upload Checklist

For each video:

- [ ] Record in 1920x1080 resolution
- [ ] Add intro/outro animations
- [ ] Include closed captions
- [ ] Add chapter markers
- [ ] Optimize for web (compress to <50MB)
- [ ] Upload to YouTube (unlisted initially)
- [ ] Add to help system database
- [ ] Test video playback in help panel
- [ ] Create thumbnail image
- [ ] Add video description with timestamps
- [ ] Tag appropriately for search

## Video Hosting

Videos should be hosted on:
1. **YouTube** (primary) - Embed in help system
2. **Vimeo** (backup) - For enterprise customers
3. **CDN** (thumbnails and previews)

## Maintenance Schedule

- Review videos quarterly for accuracy
- Update when major features change
- Re-record if UI changes significantly
- Monitor view counts and feedback
- Create new videos for new features
