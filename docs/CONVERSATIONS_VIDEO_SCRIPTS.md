## Production Notes

### Video Specifications

**Format:**
- Resolution: 1920x1080 (1080p)
- Frame rate: 30fps
- Aspect ratio: 16:9
- Format: MP4 (H.264)

**Audio:**
- Sample rate: 48kHz
- Bit rate: 192kbps
- Format: AAC
- Voiceover: Professional, friendly tone

**Branding:**
- Unbuilt logo in corner
- Neon flame color scheme
- Dark theme throughout
- Consistent typography

### Screen Recording Setup

**Browser:**
- Chrome or Firefox
- 1920x1080 window size
- Zoom level: 100%
- Clear cache before recording

**Demo Account:**
- Pre-populated with sample analyses
- Multiple conversation examples
- Various tier levels shown
- Clean, professional data

**Recording Software:**
- OBS Studio or Camtasia
- 60fps capture (export at 30fps)
- Cursor highlighting enabled
- Smooth mouse movements

### Post-Production

**Editing:**
- Add intro/outro animations
- Include captions/subtitles
- Add callout annotations
- Smooth transitions between scenes
- Background music (subtle, non-distracting)

**Captions:**
- Full transcription
- Timed accurately
- Readable font and size
- High contrast background

**Thumbnails:**
- 1280x720 resolution
- Clear, descriptive text
- Unbuilt branding
- Engaging visuals

### Distribution

**Platforms:**
